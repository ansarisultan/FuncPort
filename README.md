# Overview

FuncSpan is a modern web application built under the **FuncLexa** platform that focuses on API interaction, network simulation, and developer workflows.

The project was created to provide a clean and interactive environment where developers can explore request behavior, inspect responses, and better understand communication between frontend and backend systems.

Rather than replacing existing developer tools, FuncSpan is designed as a learning-focused project that combines modern UI with practical development concepts.

# Local Installation

Clone the repository

```bash
git clone https://github.com/ansarisultan/funcspan.git
```

Navigate to the project

```bash
cd funcspan
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
